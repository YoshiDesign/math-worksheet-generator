Worksheet Options Adjust the options for the worksheet Help

Thank you for using Version 1.0 of the new Demme Learning Worksheet Generator!

To generate a worksheet:
  • Select the Levels and Lessons dropdown.
  • Enter the desired number of problems, then select the toggle button for the chosen level(s).
  • Choose a lesson from the lesson dropdown, then click the Generate Worksheet button at the bottom of the sidebar.
To generate an answer key for your worksheet:
  • Select the Display Options dropdown.
  • Select the Show answers toggle near the bottom of the dropdown.
What if I want to include multiple lessons from one level on a worksheet?
  • Select Add Another Lesson under the lessons dropdown.
  • Select the toggle button for the additional level and choose the lesson from the lesson dropdown.
How do I adjust the appearance of the worksheet?
  • Use the sliders under the Display Options dropdown.
  • The number of problems per row, problem font size, and problem spacing can all be adjusted.
  • There are also toggles to turn on or off the name line, date line, worksheet instructions, and problem numbers.
  • A toggle to switch between horizontal and vertical equations is also included.
  • Note not all equations are made to be displayed both horizontally and vertically.
How do I make changes to a worksheet I already generated?
  • Select the desired settings, then click the Generate Worksheet button to apply them. Adjustments made using toggles or sliders will automatically apply.
  • To reset the worksheet, click the Reset button at the bottom of the sidebar.
How do I print a worksheet?
  • Select Print at the bottom of the sidebar. This will open the print preview window.
  • The Landscape preview toggle under Display Options only gives a preview of landscape orientation.
  • To print in landscape mode, landscape orientation must be selected within the print preview window.
  • If you would like to save a worksheet to print at a later time, select Save as PDF from the dropdown within the print preview window.
